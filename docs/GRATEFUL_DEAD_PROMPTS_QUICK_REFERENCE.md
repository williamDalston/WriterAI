## üåü BRAINSTORMING TEMPLATE

**If you want to create your own:**

```

```

**Examples:**
- Musician + discovers + songs open doorways
- Physicist + realizes + universe is conscious
- Wanderer + finds + crossroads of destinies

**Then add:** `"[Song name] inspired:"`
